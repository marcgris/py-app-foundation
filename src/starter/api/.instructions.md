# API Overlay Agent Instructions

This file provides agent instructions for the API overlay. It is automatically included in new projects created from this overlay.

- **Purpose:** Guide Copilot Chat and other agents on how to interact with the API overlay.
- **Instructions:**
  - Use FastAPI best practices for endpoint design.
  - Document all endpoints with OpenAPI-compatible docstrings.
  - Ensure all endpoints are covered by integration tests.
  - Follow project conventions for error handling and response models.
