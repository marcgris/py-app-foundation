# UI Overlay Agent Instructions

This file provides agent instructions for the UI overlay. It is automatically included in new projects created from this overlay.

- **Purpose:** Guide Copilot Chat and other agents on how to interact with the UI overlay.
- **Instructions:**
  - Use modern, accessible web standards for UI components.
  - Document all UI components and their props.
  - Ensure UI logic is covered by integration or E2E tests.
  - Follow project conventions for design tokens and theming.
